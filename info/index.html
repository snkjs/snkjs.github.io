司南科技社信息技术联合部 介绍
修订版本：1

1. 组织定义：
    隶属于内江六中司南科技社，进行信息技术研究、探索的部门
2. 组织目标：
    提高内江六中学生的信息技术水平及运用信息技术的能力
    参加 NOIP、科技创新大赛、中学生电子制作大赛、青少年机器人竞赛等比赛
    拓展内江六中学生课外活动
3. 组织结构：
    1)NOIP部：
        1>任务：
            - 组织、参加 NOIP，及平时的练习/刷题/讨论交流
            - 参加 NOIP/NOI
        2>要求：
            - 熟练使用 C++ 语言
            - 会使用：数据结构、递归、模拟、动态规划、贪心、图论、
                    搜索、树、堆、分治、
                    字符串、计算几何、数论、排列组合、高精度
                    等算法或结构来解决问题
    2)Web部：
        1>任务：
            - 自己设计网站 或 为学校网站开发/维护提供建议
            - 参加中学生电子制作大赛
        2>要求：
            - 掌握 HTML、JavaScript、CSS 三种语言
            - 理解 ajax 思想，会使用 ajax
            - 会建立静态网站
            - 会使用 jQuery 或 ProtoType 或 script.aculo.us 框架
            - 了解建立网站的原理 服务器提供服务的原理
            - 尝试了解使用 node.js 建立服务器
    3)编程部：
        1>任务：
            - 使用编程语言完成一定功能或开发软件
            - 参加中学生电子制作大赛
        2>开发语言：
            C/C++、C#/Java/js、VB.NET、Python、Ruby、Go、OC/Swift 等
        3>要求：
            - 能使用上述语言编写出具有某种功能的程序，可供日常使用
            - 能写出程序来参加中学生电子制作大赛
    4)安全部：
        1>任务：
            - 网络攻防，及日常安全保障、寻找漏洞等
            - 科学研究，参加科技创新大赛/中学生电子制作大赛
        2>要求：
            - 知道计算机硬件组成、软件基本原理
            - 了解计算机硬件保养知识
            - 了解基础性的软件漏洞原理
            - 了解网络攻击原理
            - 了解：电脑防病毒及其原理、Windows 系统更新原理、WinPE 原理
            - 尝试了解网络抓包等技术
            - 尝试了解如何为软件寻找漏洞
            - 尝试了解电脑防范网络攻击的技术
    5)媒体部：
        1>任务：
            - 学习：图片处理/音频制作/视频剪辑等工作
            - 参加中学生电子制作大赛
        2>要求：
            - 会使用 PhotoShop、Audition、Premiere、After Effects 等软件
    6)虚拟化部：
        1>任务：
            - 制作 3D 打印图形或制作机器人
            - 使用 Minecraft 制作地图参加中学生电子制作大赛
            - 参加青少年机器人竞赛
        2>要求：
            - 会使用 AutoCAD 或 3Done 制作可用于 3D 打印的图形
            - 会使用 Minecraft 制作高水平地图
            - 会制作机器人，使用流程图软件编写机器人程序
    7)文案部：
        1>任务：
            - 学习制作 Word/Excel/PowerPoint 文档，协助司南科技社/学校制作文档
        2>要求：
            - 熟练掌握 Word/Excel/PowerPoint 基本操作
            - 会使用 Word/Excel/PowerPoint 高级操作
            - 能制作出漂亮的 Word/Excel/PowerPoint 文档
4. 声明：
    1>所有活动均需在课外时间进行，不得占用学习时间
    2>所使用的软件、技术仅供技术交流与学习之用，不得侵犯著作者版权
